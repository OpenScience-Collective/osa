Open Science Assistant

A chat widget for HED, BIDS, and other open science tools.

Try it out

Click the chat button in the bottom-right corner to start a conversation.

Ask questions about:

  • HED (Hierarchical Event Descriptors)
  • HED validation and annotation
  • BIDS (Brain Imaging Data Structure)

Integration

To add this widget to your website:

<script src="https://your-cdn.com/osa-chat-widget.js"></script>

Or with Turnstile enabled:

<script src="https://challenges.cloudflare.com/turnstile/v0/api.js" async defer></script>
<script src="https://your-cdn.com/osa-chat-widget.js"></script>
<script>
  OSAChatWidget.setConfig({
    turnstileSiteKey: 'YOUR_SITE_KEY'
  });
</script>

API Endpoints

The widget connects to:

  • POST /hed/ask - Single question
  • POST /hed/chat - Multi-turn conversation
